# Real-Time Chat & Video Call Application

A full-stack real-time communication platform built with **Angular**, **ASP.NET Core Web API**, **SignalR**, and **WebRTC**.

---

## Overview

This project is a real-time chat application that supports:

- Private real-time messaging
- Online user presence tracking
- Peer-to-peer video calling
- Instant updates without page refresh

It demonstrates how modern web technologies can be combined to build scalable communication platforms similar to Slack, Zoom, or WhatsApp.

---

## Tech Stack

### Frontend
- Angular
- TypeScript
- RxJS

### Backend
- ASP.NET Core Web API
- SignalR

### Real-Time Communication
- SignalR (real-time messaging)
- WebRTC (peer-to-peer video calls)
